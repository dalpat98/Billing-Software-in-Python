## Features
  - Login/Register for admin and customer
  - Separate Customer/Admin Panel
  - Admin panel :
       1.modify menu(add items,delete,items, change price of items)
  - Customer panel: 
      1.view menu  
      2.order food 
  - Bill Generation based on ordered food with pre-applied taxes
